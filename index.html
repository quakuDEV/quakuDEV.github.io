An Upcoming Developer's Roadmap

Lua(u)

First, a developer needs to learn logic and the basic principles of code. Lua's simplicity and easy-to-use nature, alongside its professionalism, makes it a perfect language to learn first.

Python / JavaScript

Python is the main start of a career. It is used by many big corporations and allows a user to make games and applications. Although not the fastest, it is a great language to start getting serious. JavaScript is an alternative, as it is also really good, but is a bit harder.

C# / Any High-Level Language

C# is an example of a high-level language. When getting really good, a programmer should switch to one of these to learn. It has many more capabilities and is more impressive.

SQL + JSON

Not necessarily the last, but a developer should eventually learn these in addition to their main language. SQL and JSON allow for APIs and server work, as well as storing data.
